# Cine y Letras

> Colombia cultural heritage explorer — MinCultura open data.

Explores Colombia's cultural production: films, books, libraries, cultural spaces, and heritage sites. Data sourced from the Ministry of Culture's open APIs and datasets.

**Stack:** Vite · Vanilla JS · HTML · CSS

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Data source

[MinCultura — Ministerio de las Culturas, las Artes y los Saberes](https://www.mincultura.gov.co/areas/comunicaciones/datos-abiertos)
